# 우선순위 디스크 컨트롤러

- [문제](https://school.programmers.co.kr/learn/courses/30/lessons/42627)
  - 대기큐
      - 작업의 번호, 요청 시각, 소요 시간
  - 작업을 안 하고 있으면 우선순위가 가장 높은 작업을 실행
      - 소요시간이 짧은 것, 요청 시각이 빠른 것, 작업 번호가 작은 것

## 문제 푼 방법
- 구현
    - 맨 처음 들어온 애 시간부터 시작
    - 실행하고 끝난 시간까지 job들다 큐에 넣음
    - 남은 잡이 없으면 그 다음 최초 애로 실행
