# 풍선 터트리기

- [문제](https://school.programmers.co.kr/learn/courses/30/lessons/68646#)
    - 규칙
      1. 임의의 인접한 두 풍성 중 하나를 터트림
      2. 터진 풍선으로 빈 공간이 생기면 밀착
      3. 두 풍선 중에 더 작은 풍선을 터트리는 건 한번 밖에 못 함
## 문제 푼 방법
- 임의의 풍선이 살아남을 수 있는지 확인 방법
  - 양 쪽의 최솟값을 구함
  - 풍선의 값이 최대가 아니면 살아남을 수 있음
- 먼저 최솟값 array를 구함
- 순회하면서 각각 살아남을 수 있는지 판별하여 개수 세기
