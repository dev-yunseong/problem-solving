# 매출 하락 최소화

## 문제
[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/72416)
- 구성요소
    - 직원, 팀
    - 직원 (직원 번호, 하루 평균 매출액)
        - 직위 (팀장 또는 팀원)
- CEO
    - 직원 번호 1번 고정
    - 항상 팀장
- 나머지
    - 항상 화살표를 받음
    - 두개의 팀에 소속될 수 있다. (그럼 한쪽에는 팀원, 한쪽에선 팀장임)
- 워크숍
    - 모든 팀은 한명 이상 와야함
    - 워크숍 참석 직원의 하루 평균 매출액의 합은 최소가 되어야함


## 문제 풀 방법
- backtracking
  - current or 자식 중 선택
      - current 고르면 자식으로 넘김
      - 자식을 고르면 손자들로 넘김
- 재연산을 막기 위해 caching
