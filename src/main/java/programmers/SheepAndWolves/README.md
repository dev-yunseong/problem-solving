# 양과 늑대

## 문제
[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/92343)
- 루트에서 출발
- 각 노드를 다니면서 양을 모음
- 방문할 때마다 양과 늑대가 따라옴
- 양의 수가 항상 더 많아야한다.
- 최대 몇 마리를 얻을 수 있는 지 구하라

## 문제를 푼 방법
- backtracking
  - 방문한 곳을 넣음 stack에 넣음
  - 더 갈 곳 없으면 몇 마리 얻었는지 리턴

## 배운 점
- Back Tracking
  - 알고리즘
    1. 선택
       - 가능한 다음 선택지 중에 하나를 선택
    2. 검사
       - 이 선택으로 진행을 해봄
       - 가망 없으면 가지 치기 진행
    3. 되돌리기
       - 끝까지 가봤으면 다음 선택지 선택
  - 구현 방식: Recursive하게 구현
